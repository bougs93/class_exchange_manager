# 기본 사용법

> 💡 **유튜브 영상 가이드**: 프로그램 사용법 영상은 아래 썸네일을 클릭하여 확인하실 수 있습니다.
> 
> [프로그램 사용법 영상 보기 제작 예정](https://youtu.be/tYVXq3_XRFs)

## 시작하기

1. 왼쪽 상단 메뉴(☰)에서 '엑셀 파일 선택'을 클릭하여 시간표 파일을 불러옵니다
2. '교체 관리' 메뉴로 이동하여 교체 모드를 선택합니다

## 교체 관리

### 보기
- 시간표를 조회만 합니다
- 교체 작업을 하지 않습니다

### 1:1교체
- 두 교사의 시간을 서로 교체합니다
- 교체할 두 교사의 시간표 셀을 각각 클릭한 후 실행합니다

### 순환교체
- 여러 교사(2~3명)가 순환하며 교체합니다
- 예: A→B→C→A (A는 B의 시간, B는 C의 시간, C는 A의 시간을 받음)
- 교체할 교사들의 셀을 순서대로 클릭한 후 실행합니다

### 연쇄교체
- 직접 교체가 불가능할 때 2단계로 나누어 교체합니다
- 예: 교사 A가 월1교시 결강, 교사 B가 대체 가능하지만 교사 A가 월4교시에 수업이 있어 직접 교체 불가
  - 1단계: 교사 C 월5 ↔ 교사 A 월4 교체 (교사 A 월4 시간 비우기)
  - 2단계: 교사 A 월1 ↔ 교사 B 월4 교체 (결강 해결)
- 프로그램이 자동으로 연쇄 교체 경로를 찾아 제안합니다

### 보강
- 보강 수업을 추가합니다
- 보강할 시간과 보강할 교사를 선택한 후 실행합니다

## 결보강 문서

- 교체된 시간표를 바탕으로 다양한 문서를 생성하고 출력할 수 있습니다.

### 결보강 계획서
- 결석 날짜, 보강 날짜, 교체 정보를 선택하여 계획서를 작성합니다
- 교체 내역을 표 형식으로 확인할 수 있습니다
- PDF 문서로 출력하여 제출할 수 있습니다

### 학급 안내
- 교체 정보를 학급별로 그룹화하여 안내 메시지를 자동 생성합니다
- 복사하여 학급에 전달할 수 있습니다

### 교사 안내
- 교체 정보를 교사별로 그룹화하여 안내 메시지를 자동 생성합니다
- 복사하여 교사에게 전달할 수 있습니다

### 파일 출력
- 결보강 계획서를 PDF 파일로 출력합니다
- 학교명, 교사명 등 기본 정보를 입력합니다
- 저장 위치를 선택하고 파일명을 지정합니다

## 개인 시간표

- 특정 교사의 시간표만 필터링하여 확인합니다
- 자신의 시간표와 교체된 시간을 한눈에 볼 수 있습니다
- 교사명을 선택하면 해당 교사의 시간표만 표시됩니다
